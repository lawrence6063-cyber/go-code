它是什么 / 不是什么

DEV_SPEC §1.2

✓ cogent 是

  • CLI + Headless 执行引擎，共用一套内核
  • 在真实代码库里自主读写、执行、验证
  • 通过 MCP 消费外部工具，工具/权限/沙箱可插拔
  • fail-closed：默认不并发、默认非只读、命令默认入沙箱

✗ cogent 不是

  • 不是 IDE 插件 / 花哨 TUI 全家桶
  • 不是只会聊天的 ChatBot 套壳
  • 不内置 Remote/Bridge/Swarm 多后端分布式编排
  • 不是「信任模型输出直接 exec」的裸奔实现

自循环 Dogfood 实验成果

同机制 · 换模型 · 结局天差地别

修复前 · deepseek-reasoner

  • reviewer 输出思维链 → 3/3 被判拒
  • 拒绝即短路 → goal.verify span 0 次
  • maker cwd 幻觉（cd /workspace）
  • 结局：budget_spent 未达成，慢且贵

修复后 · deepseek-v4-pro

  • reviewer 裁决格式合规 → approved
  • verify 每轮执行 → passed=true（span 0→1）
  • 1 轮给出正确实现，相对路径无幻觉
  • 结局：achieved · 46.6s · $0.0384
关键修复经验：把客观判据封成回调（VerifyAtFunc）注入执行体，在落盘决策点之前、改动所在工作根执行—— 客观 verify 成为不可被主观裁决短路的最终硬闸门；reviewer 降级为建议反馈。详见 selfloop-fixes.html 与 selfloop-rerun-v4pro.html。
